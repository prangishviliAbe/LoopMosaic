# LoopMosaic for Elementor

![LoopMosaic Cover](assets/LoopMosaic_cover.png)

LoopMosaic is a powerful and lightweight Elementor widget designed to create flexible post grids, mosaics, and masonry layouts with advanced dynamic features. It allows you to display posts using default cards, **Elementor Loop Items**, or **JetEngine Listings**, while providing seamless integration with **JetSmartFilters** for AJAX filtering and pagination.

## 🚀 Key Features

### 🎨 Flexible Layouts
*   **Mosaic Patterns**: Choose from Classic, Metro, Masonry, Highlight First, and Uniform Grid patterns.
*   **Customization**: Control column count (3-6), gaps, padding, border radius, and overlays.
*   **Responsive**: Fully responsive grid that adapts to any screen size.

### 🔗 Integrations & Compatibility
*   **Elementor Loop Builder**: Use your custom Elementor Loop Items as card templates.
*   **JetEngine**: Fully compatible with JetEngine Listing items. Just select your listing and go.
*   **JetSmartFilters**: Native support for AJAX filtering and pagination. No page reloads required!

### ⚡ Built-in AJAX Modal System
Forget about heavy third-party popup plugins. LoopMosaic comes with a built-in, lightweight modal system.
*   **Instant Loading**: Content is fetched dynamically via AJAX.
*   **Custom Templates**: You are not limited to the default layout! Create a specific **Elementor Template** (Section or Loop Item) and use it as your popup content.
*   **Dynamic Data**: The modal automatically pulls data (Title, Image, Content, Custom Fields) for the clicked post.
*   **Gallery Support**: Option to display post gallery inside the modal.

### 🛠️ Widget Controls
*   **Query Builder**: Filter posts by type, taxonomy, order, and limit.
*   **Click Actions**:
    *   Open Single Post (Permalink)
    *   **Open Built-in Modal**
    *   None (Static Grid)
*   **Styling**: Extensive style controls for Title, Category, Overlay, and Box Shadows.

## 📦 Installation & Usage

1.  Install and activate the **LoopMosaic** plugin.
2.  Make sure **Elementor** is installed.
3.  (Optional) Install **JetEngine** or **JetSmartFilters** for advanced functionality.
4.  Open Elementor Editor and drag the **LoopMosaic Grid** widget to your page.

## 🎯 How to Use Built-in Modal with Custom Template

1.  Create a template in **Theme Builder** or **Saved Templates** (e.g., a "Single Post" view).
2.  In the LoopMosaic Widget, go to **Content > Interaction**.
3.  Set **Click Action** to **Open Built-in Modal**.
4.  Toggle **Use Custom Template** to **YES**.
5.  Select your template from the **Select Template** list.
6.  Update and preview!
